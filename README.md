# Brickbreaker 

A small Java brick breaker game originally built for BlueJ, now portable to any modern JDK.  


## Requirements
- Java 21+ 
- macOS/Windows/Linux terminal

## Quick Start

```bash
# macOS / Linux
./run.sh run Main        

# Windows
run.bat run Main
